# Cellular Mobility Research Dataset

A cellular drive test dataset supporting research into GNSS-independent positioning and mobility reconstruction from cell ID history.

---

## Overview

This repository contains a curated subset of real-world cellular drive test data, released to support research into:

- Positioning without reliance on GNSS  
- Reconstruction of movement using cell ID history  
- Mobility pattern analysis using cellular network signals  

The dataset is derived from ongoing research into telecom-native and RF-based situational awareness and is intended to support academic, industrial, and government research communities.

---

## Scope

This dataset represents:

- Real-world drive test measurements collected across terrestrial environments  
- Cellular network observations including serving and neighbouring cell information  
- Time-sequenced mobility data suitable for path reconstruction and analysis  

The dataset has been **anonymised and reduced in scope** to remove sensitive information and minimise operational risk.

---

## Intended Use

This dataset is made available to support:

- Research into resilient positioning systems  
- Development of GNSS-denied navigation techniques  
- Study of mobility inference from cellular data  
- Evaluation of signal-based localisation and tracking methods  

Use is expected to be aligned with **research, defensive, and resilience-focused applications**.

---

## Limitations

- The dataset is a **subset** of a larger research corpus  
- Data may be **incomplete, noisy, or environment-specific**  
- No ground truth positioning is guaranteed unless explicitly stated  
- The dataset is **not intended for operational deployment**
